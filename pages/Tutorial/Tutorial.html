The next task is to solve a number-placement puzzle -“Sudoku”.
The Sudoku puzzle board is a grid of 9 x 9 spaces, composed with 9 rows,
9 columns and 9 subgrids,
called “blocks” (made up of 3 x 3 spaces).
The objective is to fill the grid with digits so that each column,
each row, and each block contain all of the digits from 1 to 9.
That means, no number should appear more than once at the same column,
row or block.

The next task is to solve a color-placement puzzle -“Sudoku”.
The Sudoku puzzle board is a grid of 9 x 9 spaces, composed with 9 rows,
9 columns and 9 subgrids,
called “blocks” (made up of 3 x 3 spaces).
The objective is to fill the grid with colors so that each column, each row,
and each block contain all of the colors
according to the given information.
That means, no color should appear more than once at the same column,
row or block.

 The knapsack problem is a problem in combinatorial optimization: Given a
set of items, each with a weight and a value, determine the number of each item
to include in a collection so that the total weight is less than or equal to a
given limit and the total value is as large as possible.
